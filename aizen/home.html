Discord Bot

A customized discord bot that I have made fully myself.

Python AI JavaScript

Xerox

About

A versatile Discord bot designed to enhance server management, provide entertainment features, and integrate AI capabilities. Fully customizable and continuously updated with new features.

Commands

/hello

Sends hello to the user.

/userinfo <member>

Gets information about a chosen server member

/playerstats

Fetches player stats from Marvel Rivals

/send_commands_n8n

Manually sends the command list to the N8N webhook.

/ping

Fetches bot latency

/patchnotes

Gets the most recent patch notes from Marvel Rivals

/remind <time> <message> [recurring] [interval]

Set a reminder

/list_reminders

List your active reminders

/cancel_reminder <reminder_id>

Cancel one of your reminders

/schedule_announcement <channel> <time> <message> [recurring] [interval]

Schedule a server announcement

/list_announcements

List scheduled announcements for this server

/cancel_announcement <announcement_id>

Cancel a scheduled announcement

/math <operation> <num1> <num2>

Does simple +, -, *, / operations

/meme

Fetches random meme from r/memes

/serverinfo

Displays server information

/tempmute <member> <duration> [reason]

Temporarily mute a member

/unmute <member> [reason]

Unmute a previously muted member

/tempban <member> <duration> [reason] [delete_message_days]

Temporarily ban a member

/unban <user_id> [reason]

Unban a previously banned user

/create_role_menu <channel> <title> <description>

Create a reaction role menu

/delete_role_menu <message_id>

Delete a reaction role menu

/temprole <member> <role> <duration> [reason]

Give a user a temporary role

/removetemprole <member> <role> [reason]

Remove a temporary role from a user

/list_temp_roles

List all active temporary roles

/role_info <role>

Display information about a role

/randomnumber <min_value> <max_value>

Generates a random number between a given min and max value

/anime <anime>

Fetches information about a chosen anime from Reddit

/set_welcome <channel> [message] [enable_images]

Configure welcome message settings

/disable_welcome

Disable welcome messages

/welcome_test

Test the current welcome message

/songinfo <song>

Fetches information about a chosen song from Spotify

/valagent <agent>

Fetches information about a chosen Valorant Agent from Reddit
